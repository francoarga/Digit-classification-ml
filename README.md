# Digit-classification-ml

This project explores handwritten digit classification using machine learning techniques on the TMNIST dataset.

The project includes:

- Exploratory data analysis
- Image preprocessing
- K-Nearest Neighbors (KNN)
- Decision Trees
- Cross-validation
- Confusion matrix analysis
- Model evaluation and comparison

Dataset: TMNIST handwritten digits dataset.
Each image contains 28x28 grayscale pixels representing handwritten numbers.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Models Implemented

### K-Nearest Neighbors (KNN)
- Different values of k
- Feature subset experiments
- Accuracy comparison

### Decision Trees
- Different tree depths
- Entropy vs Gini criteria
- Cross-validation evaluation

## Results

The models achieved high classification accuracy on handwritten digit recognition tasks.

Performance was evaluated using:
- Accuracy
- Confusion matrices
- Cross-validation
